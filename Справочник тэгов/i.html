<i>

Кратко:

Тег <i> используется для изменения голоса или настроения части текста, чтобы передать его другое качество: официальное название, идиома из другого языка, транслитерация или ремарки. Для выделения важного используется <strong>, для изменения тона — <em>. По умолчанию браузеры применяют к курсивное начертание с помощью font-style: italic.
